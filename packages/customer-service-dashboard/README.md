# Customer Service Dashboard

React + TypeScript + Vite + Tailwind UI dashboard for the Customer Service System.

## Setup

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Environment Variables

Create a `.env` file:

```
VITE_API_URL=https://dartmouth-os-worker.dartmouth.workers.dev
```
